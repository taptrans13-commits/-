# ТЕХНОПРОГРЕСС

Статический сайт для подбора и расчета систем электрообогрева:

- калькулятор объема для теплого пола, кровли и трубопроводов;
- расчет количества по выбранной позиции;
- многостраничный каталог с карточками товаров по группам;
- прайс и карточки с ценами с НДС;
- заявка на почту и быстрые звонки.

Публикация выполняется через GitHub Pages workflow из ветки `main`.
